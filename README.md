# 🛠️ Tech Stack

- **Backend:** Ruby on Rails
- **Database:** PostgreSQL
- **Frontend:** HTML, CSS, Bootstrap (or Tailwind if used)
- **Server:** Puma
- **Deployment:** Render

# ⚙️ Installation (Local Setup)

```bash id="readme2"
# Clone repository
git clone https://github.com/Ayushkumarsinghyogesh/lead_manager_crm.git

# Move into project directory
cd lead-manager

# Install dependencies
bundle install

# Setup database
rails db:create
rails db:migrate

# Run server
rails server
